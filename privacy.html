Privacy Policy for Eye Protector

Effective Date: November 4, 2025

Eye Protector is a Chrome extension that helps users reduce eye strain by sending periodic screen-break reminders.

Information We Collect

- We store timer settings and usage preferences locally using Chrome Storage.
- We do not collect, transmit, or share any personal information or browsing data.

How We Use Information

Data stored locally is used solely to customize break reminders and improve user experience.

Permissions Used

  • Notifications: To display reminder pop-ups.
  • Storage: To save timer preferences locally.
  • No Remote Code: All scripts are packaged within the extension and do not load external code.
